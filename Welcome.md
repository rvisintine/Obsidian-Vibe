# Personal Life Vault

Starter Obsidian vault for personal life management. Capture media, projects, recipes, contacts, daily notes, and trips. Backed by templates and Base views.

## Getting Started

1. Open this vault in Obsidian.
2. Edit `CLAUDE.md` if using Claude Code — adjust template schemas to taste.
3. Drop a note in `Inbox/` to capture quickly. Process into the right folder later.
4. Browse `Categories/` for live Base views of each note type.
